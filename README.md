# Marketing Campaign Analysis (R)

## Overview

This project analyses marketing campaign performance using R, focusing on key metrics such as conversions, click-through rate (CTR), and return on ad spend (ROAS). The analysis explores performance across platforms, campaign types, and over time.

## Objectives

* Analyse overall marketing performance
* Compare performance across platforms and campaign types
* Evaluate efficiency using key metrics
* Visualise trends over time

## Data Preparation

* Imported dataset using tidyverse
* Cleaned and standardised column names
* Converted date fields to appropriate format

## Data Aggregation

* Calculated total spend, total conversions, and total revenue
* Computed average CTR and ROAS
* Performed grouped aggregation by platform and campaign type

## Visualisations

* Conversions by platform
* ROAS by campaign type
* Conversions over time
* Total spend over time

## Tools Used

* R
* tidyverse (dplyr, ggplot2)

## Key Insights

* Search campaigns show higher conversion efficiency, reflecting stronger user intent
* Display campaigns generate higher exposure but lower engagement
* Performance varies across platforms, with some delivering more cost-effective conversions
* Trends over time highlight fluctuations in both spend and conversion volume

## Notes

* Visualisations for conversions and spend were separated due to differences in scale
* Dataset is assumed to be representative and may not fully reflect real-world platform behaviour
